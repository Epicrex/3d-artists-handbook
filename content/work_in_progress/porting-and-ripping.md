>## Porting Skyrim SE to Blender
>- .nif
>    - Used to store model
>    - Import into outfit studio and export as fbx
>- .dds
>    - Used to store textures
>    - Open in paint.net and and save as png
>- .jslot
>    - Used to store character slider values
